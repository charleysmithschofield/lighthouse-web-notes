      To start using Flexbox, we add the display: flex property to the parent element css .parent { display: flex; } _____________________________________________________________________________________________________________________________________________________________________________________ Parent Properties: These are a list of properties we can apply to the parent element when we're using Flexbox. The parent element is also sometimes referred to as the container element. flex-direction Defines the direction that the children (also know as flex items) will appear in. We can set these to columns or rows, and dictate if the child elements should be displayed in reverse order. The flex direction also determines the main axis for the flex container. If we set to a row (the default), the main axis is horizontal; if we set the flex direction to use columns, it will be vertical. .parent { display: flex; flex-direction: row; } (would display 4 boxes: D, C, B, A from the right side of the box) .parent { display: flex; flex-direction: column; } (would display A, B, C, D in a vertical column on the left side of the box) Some other parent properties include: justify-content, determines where the flex items are along the main axis. align-items, determines how flex items are laid out on the cross axis. Child Properties Some of the child properties include: - order, which controls the order elements appear in without changing your HTML (its default value is 0) - flex-grow, dictates how much space a flex item should take up.
